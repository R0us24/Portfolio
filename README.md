# Portfolio Roustom Abdeldjalel Cherir

Portfolio interactif au thème d'un **souk oriental pixel art**, développé en React + Vite. L'interface simule une scène de souk animée où chaque boutique représente une section du portfolio (projets, compétences, CV, galerie, liens, contact).

## Concept

L'idée est de présenter mon parcours de façon originale et immersive : un personnage se déplace dans un souk du désert, et chaque échoppe ouvre une fenêtre sur une partie de mon profil. Le tout est accompagné d'animations pixel art, d'effets sonores et d'une interface bilingue français/anglais.

## Fonctionnalités

- Scène interactive animée (sable, personnage, décorations)
- Boutiques thématiques : Projets, Compétences, CV, Galerie, CTF/Hackathons, Liens, Contact
- Interface bilingue FR / EN
- Téléchargement du CV (FR & EN)
- Effets sonores et animations CSS
- Design responsive

## Stack technique

- **React** + **Vite**
- JavaScript (JSX)
- CSS personnalisé (pixel art, animations)

## Assets & Ressources

Tous les visuels, sprites, illustrations et assets graphiques (personnage, boutiques, décorations, icônes, animations) ont été **créés manuellement par moi-même avec Adobe Photoshop**. Aucun asset graphique extérieur n'a été utilisé.

## Développement assisté par IA

Le développement de ce projet a bénéficié de l'assistance de **Claude Code** (Anthropic), un outil IA en ligne de commande utilisé pour accélérer l'écriture du code, structurer les composants React et déboguer. Toute la logique, l'architecture et les choix de design restent les miens.

## Lancer le projet localement

```bash
cd souk-portfolio
npm install
npm run dev
```
